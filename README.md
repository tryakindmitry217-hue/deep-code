REST-сервис на FastAPI, который принимает с устройств JSON-статистику вида {"x": float, "y": float, "z": float}, сохраняет данные в PostgreSQL с меткой времени и идентификатором устройства, а также поддерживает ведение пользователей и назначение устройств владельцам. Аналитические задачи (расчёт минимума, максимума, количества, суммы и медианы по каждой оси) вынесены в асинхронную очередь Celery на базе Redis, что позволяет получать агрегированную и поустройственную аналитику как за заданный период, так и за всё время по устройству или по всем устройствам пользователя. Всё окружение, включая API, воркер, базу данных и Redis, упаковано в Docker Compose.
